## License

for prospect-mail Unofficial email Outlook client using Electron. It uses the Web App and wraps it as a standalone application using Electron. This only works for Microsoft/Office 365 accounts, don't use it for personal Outlook.com accounts. MIT by Julian Alarcon based on electron-outlook by Howard J https://github.com/julian-alarcon/prospect-mail/blob/main/README.md

for patches and ubtouch click package https://github.com/mathiasmb31/prospectmailut
